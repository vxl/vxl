-Select the Face:
  1. CTRL + Right Mouse click

-Extrude:
  1. Select the face to be extruded
  2. Choose "Extrude" from pop up menu
  3. Use Shift + Right Mouse
 
-Move in the camera direction: 
  1. Select one of the faces of the object
  2. Use Shift + Left Mouse

-Divide
  1. Select the Face
  2. Choose "Move" from pop up menu
  3. Select the first point on the edge of the face to be divided
  4. Select the second point on the edge of the face to be divided
